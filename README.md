# descripción del proyecto
CineLista es una aplicación web SPA que se desarrollo con React y Vite para poder  mostrar una cartelera digital de cine.  
La aplicación permite visualizar películas disponibles, buscar por título, filtrar por clasificación y destacar visualmente las películas que tienen función durante el día.

# funcionalidades principales de la pagina
 Visualización de películas en tarjetas.
 Filtro por clasificación: Todas, TE, +14 y +18.
 Búsqueda de películas por título.
 Contador de películas en función hoy.
 Etiqueta visual HOY para destacar las películas disponibles durante el día.
 Renderizado condicional cuando no existen coincidencias en la búsqueda.
 Componentes separados para mantener el código ordenado.
